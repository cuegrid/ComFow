# ComFlow

**极速转码，流转自如**

ComFlow 是一款专为 macOS 平台打造的高效视频压缩与转换工具。

---

## ✨ 核心特性 (Features)
* 🚀 **极致性能**：深度调用 macOS 原生底层能力（如 VideoToolbox 硬件加速），释放 M 系列芯片潜能。
* 🎨 **现代交互**：全新设计的可视化操作，告别繁琐的参数配置，流畅的文件加载与视图切换体验。
* 📦 **开箱即用**：独立封装，无需配置任何外部环境。
